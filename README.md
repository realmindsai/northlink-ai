# NorthLink AI

> "What does AI actually look like in *your* industry?"
> Great question. We built 21 answers.

**Live site:** [realmindsai.github.io/northlink-ai](https://realmindsai.github.io/northlink-ai/)

## What is this?

A static demo site built for the **NorthLink AI Summit** (25 March 2026, Broadmeadows Town Hall). Someone walks up to our stand, picks their industry, and watches an AI tool come to life on screen — complete with fake data, slick animations, and zero actual machine learning.

Yes, it's all smoke and mirrors. But *very convincing* smoke and mirrors.

## Tech Stack

- **HTML, CSS, vanilla JS.** That's it. No React. No Next. No 47 node_modules folders.
- Google Fonts (Epilogue) and Chart.js/Leaflet via CDN
- Hosted on GitHub Pages because we're not monsters

## Running Locally

```bash
cd northlink-ai
python3 -m http.server 8000
# open http://localhost:8000
```

Or just double-click `index.html`. We won't judge.
